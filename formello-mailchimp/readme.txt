== Description ==

This is an extension of Formello taht adds Mailchimp action to your forms. 

With this plugin you can automatically subscribe email to your preferred list on Mailchimp.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/formello-mailchimp` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
